# 🚀 Web监控系统快速启动

## 第一次使用

### 1. 安装Flask
```bash
conda activate SmartPlanter
pip install -r Web/requirements.txt
```

### 2. 启动服务器
```bash
cd Web
python app.py
```

### 3. 打开浏览器
访问: **http://localhost:5000**

## 网页功能

### 📊 湿度监控
- 自动显示过去24小时湿度折线图
- 显示当前湿度、平均湿度、数据点数
- 每5秒自动刷新

### 💬 对话控制
- **绿色按钮**: 开启对话
- **红色按钮**: 关闭对话  
- **蓝色按钮**: 手动刷新数据

### 📝 活动日志
- 蓝色 = 系统消息
- 浅蓝 = 用户说的话
- 绿色 = 植物回复
- 粉色 = 动作识别

## 测试数据

首次启动会自动生成24小时示例数据，可以：
- 查看折线图效果
- 测试对话控制按钮
- 查看日志显示

## 连接传感器

后续可以通过API上传真实湿度数据：

```python
import requests

# 每小时上传一次
humidity = 75.5  # 从传感器读取
requests.post('http://localhost:5000/api/humidity/add',
              json={'humidity': humidity})
```

## 常见问题

**Q: 网页打不开？**
A: 确保Flask服务器正在运行，检查5000端口

**Q: 数据不更新？**  
A: 点击刷新按钮，或检查浏览器控制台

**Q: 图表不显示？**
A: 需要网络连接加载Chart.js库
