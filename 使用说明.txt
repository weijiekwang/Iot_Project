智能盆栽交互系统 - 快速使用指南
===========================================

文件结构：
  Iot_Project/
  ├── smart_plant_system.py    # 完整系统（动作+语音）
  ├── Gesture/                 # 动作识别模块
  └── Voice/                   # 语音识别模块

运行方法：
  1. 完整系统: python smart_plant_system.py
  2. 只测试动作: python Gesture/gesture_recognition.py
  3. 只测试语音: python Voice/voice_recognition.py

安装依赖：
  pip install -r Gesture/requirements.txt
  pip install -r Voice/requirements.txt

详细说明请查看各文件夹内的README.md
